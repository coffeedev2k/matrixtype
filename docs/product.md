# Product Vision

MatrixType teaches blind typing by binding each typed character to a spoken body command. The learner says the command aloud, attends to the physical movement, and presses the key only after that command is clear.

V1 is intentionally small: browser-only runtime, Russian and English keyboard layouts, custom text, and open command maps that anyone can study or improve.

The product is not a speed-first typing game. It is a precision-first learning tool: speak the command, feel the movement, press the right key, and repeat until the mapping becomes embodied.

The welcome screen should meet new learners in their browser language when that language is supported. This keeps the entry barrier low before the learner spends effort on the actual skill.
