# Product Vision

MatrixType teaches blind typing by binding each typed character to a spoken body command. The learner says the command aloud, attends to the physical movement, and presses the key only after that command is clear.

V1 is intentionally small: browser-only runtime, independent interface language and keyboard layout, custom text, browser-local statistics, and open command maps that anyone can study or improve.

The product is not a speed-first typing game. It is a precision-first learning tool: speak the command, feel the movement, press the right key, and repeat until the mapping becomes embodied.

The welcome screen should meet new learners in their browser language when that language is supported. This keeps the entry barrier low before the learner spends effort on the actual skill.

Custom text keeps its case. Uppercase letters are supported as Shift-derived symbols: the learner still follows the body command for the base key, with an added Shift phrase.

The trainer tracks active practice time, typed characters, typed words, and word speed locally in the browser. If the learner stops typing for more than 5 seconds, practice time pauses. The current product target is 3 hours of active practice time.
