# Language Layout Rollout

V1 layout rollout is keyboard-first, and the interface now supports the same broad language set for onboarding: `ru`, `en`, `es`, `pt`, `fr`, `de`, `it`, `pl`, `uk`, `tr`, `nl`, `cs`, and `sk`.

| Language | Layout id | Reference direction | Supported input in V1 | Deferred | Status |
| --- | --- | --- | --- | --- | --- |
| English | `en-us-qwerty` | US QWERTY | Latin letters, space, basic punctuation, Shift uppercase | US-International dead keys | Done |
| Russian | `ru-qwerty` | Russian ЙЦУКЕН | Cyrillic letters, space, basic punctuation, Shift uppercase | Variant punctuation | Done |
| Spanish Latin America | `es-latam-qwerty` | Microsoft `KBDLA` / Keyboard Layout Info | Latin letters, `ñ`, space, basic punctuation, Shift uppercase, acute dead keys, diaeresis dead key for `ü` | More punctuation | Done |
