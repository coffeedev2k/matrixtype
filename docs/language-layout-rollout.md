# Language Layout Rollout

V1 layout rollout is keyboard-first, and the interface now supports the same broad language set for onboarding: `ru`, `en`, `es`, `pt`, `fr`, `de`, `it`, `pl`, `uk`, `tr`, `nl`, `cs`, and `sk`.
