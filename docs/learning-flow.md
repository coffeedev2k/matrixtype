# Learning Flow

1. Choose the interface language and the keyboard layout independently.
2. Read the current character and spoken command.
3. Say the command aloud.
4. Do not look at the keyboard.
5. Press the matching key slowly and accurately.
6. Advance only after the correct input.

Wrong input flashes the screen and keeps the cursor in place.

Uppercase letters are trained as the same base key plus Shift. For example, `S` uses the command for `s` with an added Shift phrase. The active hand and finger still show the base key; Shift is not shown as a separate hand action in V1.

Command grammar:

- The first part names the working hand: left or right.
- The second part names the working finger number. Fingers are counted from the thumb, but the thumb itself is not counted.
- The third part names the movement of that target finger, such as in place, up, down, right, or a multi-step direction.
- The learner should say all three parts aloud before pressing the key.

Finger numbering:

- Thumb is not counted.
- First finger is the one next to the thumb.
- Left hand visual order is `4, 3, 2, 1`.
- Right hand visual order is `1, 2, 3, 4`.
